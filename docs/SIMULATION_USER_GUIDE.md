## Getting Started

### 1. Access the Simulation

1. Log in to your BTEC Smart Platform account
2. Navigate to **Simulation** from the sidebar menu
3. You'll see three tabs: **Run Simulation**, **History**, and **Compare**

### 2. Setup Your Business

Fill in the following fields to configure your business simulation:

#### **Student Name** (Required)
Your identifier for tracking simulation results. This name will appear in the history and leaderboards.

#### **Project Type** (Required)
Choose the focus of your business:
- **Technology**: Best for tech-focused projects, gets +20 tech score bonus
- **Finance**: Balanced approach, gets +10 tech score bonus
- **Marketing**: Best for market-facing projects, gets +15 market score bonus

#### **Budget** (Required)
Select your initial funding level:
- **Low**: $20,000 starting capital (smaller building in 3D view)
- **Medium**: $50,000 starting capital (mid-size building) - **Recommended**
- **High**: $100,000 starting capital (skyscraper building)

üí° *Higher budget = better financial stability and growth potential*

#### **Marketing Strategy** (Required)
How you'll reach customers:
- **Social Media**: +30% market score - **Most Effective**
- **Influencer**: +25% market score
- **Traditional**: +10% market score (TV, radio, print)

üí° *Social media marketing reaches the most customers in the digital age*

#### **Complexity Level** (Required)
Project sophistication:
- **Basic**: Good for learning
- **Intermediate**: Balanced complexity - **Recommended**
- **Advanced**: +10 tech score, best for experienced users

#### **Cybersecurity Level** (Optional)
Data protection investment:
- **Basic**: +3 finance score, minimal protection
- **Advanced**: +10 finance score, good protection - **Recommended**
- **Enterprise**: +15 finance score, maximum protection (-90% risk)

üí° *Better cybersecurity reduces financial risks significantly*

#### **AI Integration** (Optional)
Artificial Intelligence adoption:
- **None**: No AI benefits
- **Basic**: +8 tech score - **Good Starting Point**
- **Advanced**: +15 tech score - **Best Performance**

üí° *Advanced AI integration boosts efficiency by 15%*

### 3. Run the Simulation

1. Click **"Generate Virtual World"** button
2. Watch the 3D environment load with your building
3. The simulation will progress through 12 months automatically
4. Live metrics update in real-time (top-left corner)
5. After completion, view your detailed results

## Understanding the 3D View

### Camera Controls

**Mouse Controls:**
- **Left Click + Drag**: Rotate the view around the building
- **Right Click + Drag**: Pan the camera (move side to side)
- **Scroll Wheel**: Zoom in and out

**Control Panel (Bottom-Right):**
The floating control panel provides preset camera angles:

- **Front View**: Face the building from the front
- **Top View**: Bird's eye view from above
- **Side View**: View from the side
- **Isometric View**: 45-degree angle view - **Default**
- **Reset**: Return to the default camera position
- **Screenshot**: Download a PNG image of the current view
- **Fullscreen**: Expand to full screen mode

üí° *Hover over buttons to see tooltips*

### Building Types

Your budget determines which building appears:

#### **Skyscraper** (High Budget - $100k+)
- Large 5-story metallic tower
- Cyan/teal glow with illuminated windows
- Represents a major enterprise
- Position: Center of the 3D grid

#### **Modern Building** (Medium Budget - $50k)
- Mid-size 3-story blue building
- 6 illuminated windows
- Represents a growing startup
- Good balance of size and efficiency

#### **Garage** (Low Budget - $20k)
- Small 1.5-story structure
- Red pyramid roof
- Represents a bootstrap startup
- Lower growth potential but manageable

### Environment Elements

- **Grid**: Cyan-colored grid helps with spatial reference
- **Ground**: Dark platform that receives shadows
- **Sky**: Dynamic sky with moving sun
- **Lighting**: Multiple light sources create realistic shadows
- **Shadows**: Building casts dynamic shadows on the ground

### Live Data Viewer (Top-Left)

Shows real-time information during simulation:

- **Current Month**: Progress indicator (Month X/12)
- **Elapsed Time**: How long the simulation has been running
- **Tech Performance**: Current technology efficiency score
- **Finance Performance**: Current financial stability score
- **Market Performance**: Current market share score
- **Market Volatility**: Risk level percentage
- **Recent Activity**: Competitor actions and market events

üí° *Click the eye icon to show/hide the data viewer*

## Troubleshooting

### 3D View Not Loading

**Problem**: Black screen or frozen 3D view

**Solutions**:
1. Check browser compatibility
   - ‚úÖ Recommended: Chrome, Firefox, Edge
   - ‚ö†Ô∏è Safari may have issues
   - ‚ùå Internet Explorer not supported

2. Enable WebGL
   - Visit: `chrome://gpu` (Chrome) or `about:support` (Firefox)
   - Check if WebGL is enabled
   - Update graphics drivers if needed

3. Clear browser cache
   - Press `Ctrl + Shift + Delete`
   - Clear cached images and files
   - Reload the page

4. Disable browser extensions
   - Ad blockers may interfere
   - Try incognito/private mode

### Can't Save Simulation

**Problem**: "Failed to save simulation" error

**Solutions**:
1. Check internet connection
2. Verify you're logged in (check top-right corner)
3. Try refreshing the page
4. Check localStorage quota (not full)
5. Try a different browser

üíæ **Note**: Simulations save locally even if server fails

### Performance Slow

**Problem**: Lag, stuttering, or slow response

**Solutions**:
1. Close other browser tabs
2. Close unused applications
3. Reduce graphics quality:
   - Hide the 3D data viewer
   - Use simple camera views
   - Avoid frequent zoom/pan

4. Clear simulation history:
   - Delete old simulations
   - Keep only last 10-20

5. Update your browser to the latest version

### Results Don't Make Sense

**Problem**: Unexpected scores or outcomes

**Remember**:
- Market volatility adds randomness (0-50%)
- Results vary between runs
- Random factors simulate real-world uncertainty
- Try running the same setup multiple times
- Average results across 3-5 runs for patterns

## Keyboard Shortcuts

Speed up your workflow with these shortcuts:

### Navigation
- `Tab`: Move between form fields (in setup)
- `Enter`: Submit simulation (in setup)
- `Esc`: Exit fullscreen or close modals

### 3D View
- `R`: Reset camera to default position
- `F`: Toggle fullscreen mode
- `H`: Toggle history panel visibility
- `C`: Toggle controls panel visibility

### Interface
- `Ctrl + S`: Save current simulation (if supported)
- `Ctrl + P`: Print/export current view
- `Ctrl + F`: Focus search bar (in history)

### Browser
- `F5` or `Ctrl + R`: Reload page
- `Ctrl + +`: Zoom in
- `Ctrl + -`: Zoom out
- `Ctrl + 0`: Reset zoom

## Frequently Asked Questions

**Q: How long does a simulation take?**
A: About 6-10 seconds for the 12-month simulation

**Q: Can I pause a running simulation?**
A: No, but simulations are quick. Just let it complete.

**Q: How many simulations can I save?**
A: Up to 50 locally. Older ones are automatically removed.

**Q: Do simulations work offline?**
A: Yes, but you need internet to save to the server.

**Q: Can I edit a completed simulation?**
A: No, but you can copy the settings and run a new one.

**Q: What happens if I close the tab during simulation?**
A: The simulation will stop. Start a new one when you return.

**Q: Can teachers see my simulations?**
A: Only if you export and share the report.

**Q: How accurate is the economic model?**
A: It's simplified for education, not a real business predictor.
